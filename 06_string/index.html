字符串 —— 文本信息的载体

1. 什么是字符串 —— 三种写法，任你选择

字符串就是一串字符，像一串糖葫芦——每个字是一颗山楂，串在一起就是完整的信息。

Python 提供三种引号写法，根据场景灵活选择。

三种表示法

'Hello' # 单引号 —— 最简洁 "It's Python" # 双引号 —— 内容含单引号时用 '''第一行 第二行 第三行''' # 三引号 —— 多行文本

动态演示：每种写法何时用

1s1 = 'Hello Python' # 单引号：最常用，简短字符串
2s2 = "It's a nice day" # 双引号：内容含撇号（'）时避免冲突
3s3 = '''第一行
4第二行
5第三行''' # 三引号：保留换行格式
6
7print(s1)
8print(s2)
9print(s3)

2. 索引与切片 —— 精准取出想要的字符

字符串像一排带编号的储物柜，每个字符占一个格子。

正向索引从 0 开始；反向索引从 -1 开始（倒数第一）。

切片 [start:end:step] = 从 start 开始，到 end 之前停止，步长为 step。

索引与切片语法

s[i] # 取第 i 个字符（从0开始） s[start:end] # 切片：取 [start, end) 范围 s[start:end:step] # 带步长的切片 s[::-1] # 反转字符串

动态演示：可视化索引与切片

s[ : ]

3. 基本操作 —— +、*、in

+ = 粘合两段文字，* = 重复多遍，in = 检查"在不在里面"。

3.1 拼接 —— "+" 把字符串粘起来

1a = "Hello"
2b = " World"
3result = a + b
4print(result)

3.2 重复 —— "*" 让字符串自我复制

1word = "Hi"
2result = word * 3
3print(result)

3.3 成员判断 —— "in" 检查是否存在

1text = "Python"
2result1 = "Py" in text
3result2 = "tho" in text
4result3 = "abc" in text
5print(result1, result2, result3)

4. 综合演示 —— 完整的字符串处理

用户输入姓名 → 拼接问候语 + 取首字母 + 检查是否含某字符：

1name = input("请输入你的名字：")
2greeting = "你好，" + name + "！"
3first_char = name[0]
4has_a = "a" in name or "阿" in name
5print(greeting)
6print(f"名字首字母是：{first_char}")
7print(f"名字含'a'或'阿'：{has_a}")
