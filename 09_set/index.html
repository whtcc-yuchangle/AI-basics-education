集合 —— 不重复的无序宝库

1. 什么是集合 —— 装满"唯一"的袋子

类比：集合就像一袋不同颜色的弹珠—— 每颗弹珠的颜色在袋子里只有一颗（自动去重），而且袋子里的弹珠没有固定的排列顺序（无序）。

用花括号 { } 包起来，逗号分隔，就创建了一个集合。空集合必须用 set() 来创建（{} 是空字典，不是空集合！）。

基本语法

# 创建集合的几种方式 fruits = {"apple", "banana", "cherry"} # 花括号创建 nums = {1, 2, 2, 3, 3, 3} # 重复元素会被自动去重 → {1, 2, 3} empty_set = set() # 空集合，必须用 set() chars = set("hello") # 从字符串创建 → {'h','e','l','o'}

⚠️ 两个关键特性：

1. 无序 —— 集合没有"第 0 个"、"第 1 个"的概念，不能通过索引访问

2. 去重 —— 相同的元素只能出现一次，重复的会自动被忽略

动态演示：集合的创建与自动去重

1fruits = {"apple", "banana", "cherry"}
2nums = {1, 2, 2, 3, 3, 3} # 重复的数字
3chars = set("hello") # 字符串 → 拆成字符集合
4
5print(fruits)
6print(nums)
7print(chars)

2. 集合的运算 —— 并、交、差、对称差

类比：把集合想象成两个圈圈，元素散落在里面。并集是把两个圈合并（所有元素），交集是圈圈重叠的部分（共有元素），差集是只在你这边不在那边的元素，对称差是两边不重叠的部分加起来。

语法速览

A = {1, 2, 3, 4} B = {3, 4, 5, 6} A | B → {1,2,3,4,5,6} # 并集（union）：所有元素，重复的只算一次 A & B → {3, 4} # 交集（intersection）：共有的元素 A - B → {1, 2} # 差集（difference）：只在 A 中，不在 B 中 A ^ B → {1,2,5,6} # 对称差（symmetric_difference）：去掉共有的

动态演示：四种运算一目了然

1A = {1, 2, 3, 4}
2B = {3, 4, 5, 6}
3print(A | B) # 并集
4print(A & B) # 交集
5print(A - B) # 差集
6print(A ^ B) # 对称差

3. 集合的增删操作 —— 往袋子里拿放东西

类比：集合就像一个可以随时增减物品的收纳袋 —— 你可以往里放新东西（add），也可以往外扔东西（discard / remove），还可以随机掏出一个（pop），或者直接清空整个袋子（clear）。

增删方法语法

s = {1, 2, 3} s.add(4) → {1, 2, 3, 4} # 添加元素（已存在则什么都不做） s.discard(2) → {1, 3, 4} # 安全删除：元素不存在不会报错 s.remove(1) → {3, 4} # 强制删除：元素不存在会报 KeyError s.pop() → 随机弹出一个 # 删除并返回任意一个元素 s.clear() → set() # 清空所有元素

动态演示：看着集合"动"起来

1s = {1, 2, 3}
2s.add(4) # 添加新元素
3s.discard(2) # 安全删除
4s.remove(3) # 强制删除
5popped = s.pop() # 随机弹出一个
6print(s)
7print("弹出:", popped)
8s.clear() # 清空集合
9print("清空后:", s)

4. 集合的常用操作 —— 统计、复制、转换

类比：除了增删元素，集合还有一套实用工具：数一数袋子有几个东西（len）、检查某个东西在不在里面（in / not in）、复制一个完全一样的袋子（copy）、或者把一列东西变成袋子（set() 转换）。

动态演示：len / in / not in / copy / set()

1s = {3, 1, 4, 1, 5, 9}
2print(len(s)) # 集合大小
3print(4 in s) # 成员判断
4print(10 not in s) # 不在里面？
5t = s.copy() # 复制集合
6lst = [1, 2, 2, 3, 3] # 列表有重复
7print(set(lst)) # set() 转换 → 去重！

5. 集合的应用 —— 包含关系与数据去重

生活应用：

• 数据去重：把列表转成集合，重复数据自动消失！list(set(数据)) 是 Python 去重的经典写法。

• 包含关系：检查一个集合是不是另一个的子集/超集，适合做权限检查、范围判断等。

• 共同好友：用交集 & 快速找出两个人的共同好友。

子集与超集

A = {1, 2, 3, 4} B = {1, 2} B <= A → True # B 是 A 的子集（B 的所有元素都在 A 中） A >= B → True # A 是 B 的超集（A 包含 B 的所有元素） A > B → True # A 是 B 的真超集（A 包含 B，而且 A 还有更多元素）

动态演示：包含关系与去重

1all_students = {"小明", "小红", "小刚", "小丽"}
2group_a = {"小明", "小丽"}
3print(group_a <= all_students) # A组的都在全体里吗？
4print(all_students >= group_a) # 全体包含A组吗？
5
6data = [1, 2, 2, 3, 3, 3, 4] # 原始数据有重复
7unique = list(set(data)) # 集合去重后再转回列表
8print("去重后:", unique)

6. 综合演示 —— 两个班级的选课系统

用集合来管理两个班级的课程选修情况，体验 创建 → 运算 → 判断 → 去重 的完整流程。

1class_a = {"小明", "小红", "小刚", "小丽"} # A班学生
2class_b = {"小刚", "小丽", "小强", "小美"} # B班学生
3
4# 两个班都选课的学生（交集）
5common = class_a & class_b
6print("共同选课:", common)
7
8# 只在A班选课的学生（差集）
9only_a = class_a - class_b
10print("仅A班:", only_a)
11
12# 所有选课学生（并集，自动去重）
13all_students = class_a | class_b
14print("总人数:", len(all_students))
15print("名单:", all_students)
