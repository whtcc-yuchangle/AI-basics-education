循环语句

Part 1：while 循环 — "当……的时候，就一直做……"

生活类比：喝汤

汤烫 → 吹一勺 → 凉一点 → 还烫？→ 再吹 → …… → 不烫了 → 大口喝！

while = 不知道具体次数，只知道"条件满足就继续"

while 条件: # 循环体 # 必须有让条件趋向False的语句！
count = 1 while count <= 3: print(count) count += 1 # 关键！最终让条件为False print("结束")

while 案例一：简单计数器

n = int(input("请输入N：")) count = 1 while count <= n: print(count) count += 1

动态演示

1

while 案例二：猜数字游戏

secret = random.randint(1, 100) guess = 0 while guess != secret: guess = int(input("请输入猜测：")) if guess < secret: print("太小了！") elif guess > secret: print("太大了！") else: print("恭喜！")

while 案例三：计算阶乘

n = int(input("请输入：")) result = 1; i = 1 while i <= n: result *= i i += 1 print(f"{n}! = {result}")

动态演示

1

Part 2：for 循环 — "对每个元素做同样的事"

生活类比：检查书包

书包里有4件东西 → 一件一件拿出来检查 → 拿完了 → 停止

for = 知道要处理几个东西，逐个遍历

for 变量 in 可迭代对象: # 对每个元素执行的操作
range(start, stop, step)
参数含义默认
start从哪开始（包含）0
stop到哪停止（不包含）必填
step步长1

while vs for

选 while

不知道具体次数，"条件满足就继续"：猜数字、等用户输入"退出"

选 for

知道处理几个东西：遍历列表、计算1到100的和

for 案例一：遍历列表

fruits = ["苹果", "香蕉", "橙子", "葡萄"] for fruit in fruits: print(f"我喜欢吃{fruit}") print("全部介绍完毕！")

动态演示

for 案例二：累加求和

n = int(input("请输入N：")) total = 0 for i in range(1, n + 1): total += i print(f"1到{n}的和是: {total}")

动态演示

0

for 案例三：打印金字塔

n = int(input("请输入层数：")) for i in range(1, n + 1): spaces = " " * (n - i) stars = "*" * (2 * i - 1) print(spaces + stars)
