分支语句

分支语句的基本概念

生活类比：出门看天气

单分支："如果下雨，就拿伞。" — 只有一种情况行动

双分支："如果下雨，就拿伞；否则，戴帽子。" — 两条路走一条

多分支："大雨→雨衣，小雨→打伞，阴天→带伞，晴天→帽子" — 逐个判断

分支 = 程序在路口做选择，条件就是路标

三种分支语法

单分支 if

if 条件: # 条件为True时执行

双分支 ifelse

if 条件: # 条件为True时执行 else: # 条件为False时执行

多分支 ifelifelse

if 条件1: # 条件1为True执行 elif 条件2: # 条件2为True执行 elif 条件3: # 条件3为True执行 else: # 都不满足时执行

比较运算符速查

运算符含义示例结果
==等于5 == 5True
!=不等于5 != 3True
>大于10 > 6True
<小于3 < 8True
>=大于等于5 >= 5True
<=小于等于4 <= 9True

案例一：单分支 — 投票资格 if

age = int(input("请输入年龄：")) if age >= 18: print("你有投票权！")

案例二：双分支 — 奇偶数 ifelse

num = int(input("请输入整数：")) if num % 2 == 0: print(f"{num}是偶数") else: print(f"{num}是奇数")

案例三：多分支 — 成绩等级 ifelifelse

score = int(input("分数(0-100)：")) if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" elif score >= 60: grade = "D" else: grade = "F" print(f"等级：{grade}")
