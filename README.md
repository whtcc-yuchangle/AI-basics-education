# Python 基础知识教育演示

一个面向初学者的 Python 基础语法教学项目，通过交互动画和可视化演示讲解核心概念。

## 模块

| 模块 | 内容 |
|------|------|
| **输入输出** | `print()` 多参数/格式化、`input()` 类型转换、缩进可视化 |
| **运算符** | 算术运算、赋值运算、比较运算、逻辑运算、闰年综合案例 |
| **分支语句** | 单分支/双分支/多分支判断、流程图动画 |
| **循环** | `while` 计数器/猜数字/阶乘、`for` 列表遍历/累加求和/金字塔 |

## 使用

直接在浏览器中打开任意 `.html` 文件即可，无需安装或构建。

```bash
# 克隆仓库
git clone https://github.com/whtcc-yuchangle/AI-basics-education.git

# 用浏览器打开首页
open index.html
```

## 技术栈

纯 HTML + CSS + JavaScript，无外部依赖，每个模块一个独立文件。
