# Python 基础知识教育演示

一个面向初学者的 Python 基础语法教学项目，通过交互动画和可视化演示讲解核心概念。

## 模块

| 模块 | 内容 |
|------|------|
| **输入输出** | `print()` 多参数/格式化、`input()` 类型转换、缩进可视化 |
| **运算符** | 算术运算、赋值运算、比较运算、逻辑运算、闰年综合案例 |
| **分支语句** | 单分支/双分支/多分支判断、流程图动画 |
| **循环** | `while` 计数器/猜数字/阶乘、`for` 列表遍历/累加求和/金字塔 |
| **函数** | `def` 定义与调用、参数传递、`return` 返回值、综合演示 |
| **字符串** | 三种表示法、索引与切片、`+`/`*`/`in` 基本操作、综合演示 |
| **列表** | 创建访问、增删操作、排序反转、统计查找、综合案例 |
| **元组** | 创建方式、索引切片、不可变性、合并重复、成员判断、元组解包 |
| **集合** | 自动去重、并交差补、增删操作、成员判断、包含关系、数据去重 |
| **字典** | 键值映射、增删改查、keys/values/items、成员判断、综合案例 |
