输入输出与基本语法

1. print() —— 程序的"嘴巴"

生活类比

想象你和一个机器人对话：

你想让机器人说一句话 → 用 print() → 机器人就把括号里的内容"说"出来

print() = 让程序开口说话的工具

基本语法

print(内容) # 输出并自动换行 print(a, b, c) # 多个内容，默认空格分隔 print("Hi", end="!") # 自定义结尾（默认 \n） print(1, 2, 3, sep=" - ") # 自定义分隔符（默认空格）

格式化输出的三种方式

name = "小明" age = 12 # 方式1：f-string（推荐！） print(f"我叫{name}，今年{age}岁") # 方式2：format() 方法 print("我叫{}，今年{}岁".format(name, age)) # 方式3：逗号拼接 print("我叫", name, "，今年", age, "岁")

2. input() —— 程序的"耳朵"

生活类比

你想让机器人听你说话 → 用 input() → 程序暂停等你敲键盘 → 你按回车后记住你输入的内容

input() = 让程序接收用户信息的耳朵

变量 = input() # 无提示，等待输入 变量 = input("请输入：") # 带提示文字

重要：input() 返回的永远是字符串！

age = int(input("请输入年龄：")) # int() 转整数 height = float(input("请输入身高：")) # float() 转小数

动态演示：和程序对话

3. 缩进 —— Python 的"骨架"

生活类比

缩进就像写文章的段落格式：大标题顶格，正文缩进。

Python 靠缩进区分代码块的层级，不用大括号 {}！

缩进 = Python 区分代码块的唯一方式

规则说明
统一空格通常用 4个空格（不要混用Tab！）
同级对齐同一层级代码必须对齐到同一列
冒号开头if/for/while/def 等以冒号结尾，下一行缩进
可嵌套缩进里可再缩进，每层多4个空格

生活类比

注释就像课本上的批注——写给人看的，Python 解释器完全忽略它们。

注释 = 给自己和他人留的便条，不参与执行
