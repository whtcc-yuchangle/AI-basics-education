字典 —— 键值对应的映射宝典

1. 什么是字典 —— "根据名字找东西"

类比：字典就像一本电话簿 —— 你通过"姓名"（键）就能找到"电话号码"（值），一一对应，这就是 映射！

用花括号 { } 包起来，每个条目用 键: 值 的形式，逗号分隔，就创建了一个字典。

基本语法

# 创建字典的几种方式 student = {"name": "小明", "age": 18, "score": 92} # 花括号 + 键:值 empty = {} # 空字典 by_dict = dict(name="小明", age=18) # dict() 关键字参数创建 pairs = dict([("a",1), ("b",2)]) # 从键值对列表创建

⚠️ 重要特性：字典的键必须是不可变类型（字符串、数字、元组），不能是列表或集合！值可以是任意类型。

动态演示：字典的"组装"过程

2empty = {}
3by_dict = dict(name="小红", age=19)
4
5print(student)
6print(empty)
7print(by_dict)

2. 字典的访问 —— 通过键找到值

类比：字典就像储物柜 —— 每个柜门上都贴着标签（键），你找到标签就能拿到里面的东西（值）。

用 d[键] 快速取值；用 d.get(键) 安全取值（不存在返回 None，不会报错）。

访问语法

d = {"name": "小明", "age": 18} d["name"] → "小明" # 用方括号 + 键名，快速直接 d.get("age") → 18 # get() 方法，安全可靠 d.get("sex", "未知") → "未知" # 键不存在时返回默认值

动态演示：精准找到每一个值

1d = {"name": "小明", "age": 18, "score": 92}
2print(d["name"]) # 通过键取值
3print(d.get("age")) # get 安全取值
4print(d.get("sex", "未知")) # 默认值

3. 字典的增删改 —— 随用随调

类比：字典就像一个动态的记账本 —— 可以记上新条目（添加），改掉旧数据（修改），撕掉不要的页（del / pop）。

增删改语法

d = {"name": "小明", "age": 18} d["score"] = 92 → {... "score": 92} # 添加新键值对 d["age"] = 19 → {... "age": 19} # 修改已有键的值 del d["score"] → {...} # 删除指定键值对 d.pop("name") → "小明" # 弹出并返回被删除的值

动态演示：字典的增删改

1d = {"name": "小明", "age": 18}
2d["score"] = 92 # 添加新键
3d["age"] = 19 # 修改值
4del d["score"] # 删除键
5popped = d.pop("name") # 弹出键
6print(d)
7print("被弹出:", popped)

4. 字典的常用操作 —— keys / values / items / len / in

类比：字典还附带了多种查看方式：只看所有标签（keys）、只看所有物品（values）、标签和物品一起看（items）、数一数有几格（len）、检查某个标签是否存在（in）。

动态演示：keys / values / items / len / in

1d = {"name": "小明", "age": 18, "score": 92}
2print(d.keys()) # 所有键
3print(d.values()) # 所有值
4print(d.items()) # 所有键值对
5print(len(d)) # 字典大小
6print("name" in d) # 键是否存在
