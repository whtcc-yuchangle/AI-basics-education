Python 运算符

1. 算术运算符 + - * / // % **

生活类比

加减乘除和数学一样，但 Python 的除法有两种：

/ 精确除法，// 地板除（只要整数部分），% 取余数，** 是乘方

运算符名称示例结果说明
+加法5 + 38—
-减法5 - 32—
*乘法5 * 315用 * 不是 ×
/除法5 / 22.5始终返回 float
//地板除5 // 22向下取整
%取余（模）5 % 215 ÷ 2 = 2 余 1
**幂运算5 ** 2255 的 2 次方

2. 赋值运算符 = += -= *= /=

生活类比

赋值就像往盒子里放东西：

x = 5 → 把 5 放进贴有 "x" 标签的盒子；x += 3 → 盒子里已有数，再加 3

运算符等价写法示例 (x=10)x 变成
=基本赋值x = 1010

-=x = x - nx -= 37

5. 综合演示：判断闰年

规则：能被4整除且不能被100整除，或者能被400整除

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0): print(f"{year}年是闰年") else: print(f"{year}年不是闰年")
