列表 —— 有序数据的容器

1. 什么是列表 —— 给数据排个队

类比：列表就像一列火车 —— 每节车厢装一个数据，车厢按 0, 1, 2... 的顺序编号。你可以把任何东西放进车厢：数字、文字、甚至另一个列表！

用方括号 [ ] 把数据包起来，逗号分隔，就创建了一个列表。

基本语法

# 创建列表的几种方式 nums = [1, 2, 3, 4, 5] # 数字列表 fruits = ["apple", "banana"] # 字符串列表 mixed = [1, "hello", 3.14] # 不同类型混搭 empty = [] # 空列表，以后可以往里加 nested = [[1,2], [3,4]] # 列表里套列表（嵌套）

动态演示：看看列表是怎么"组装"起来的

1fruits = ["apple", "banana", "cherry"]
2nums = [10, 20, 30]
3mixed = [1, "hi", True]
4
5print(fruits)
6print(nums)
7print(mixed)

2. 列表的访问 —— 精准找到每一个元素

类比：列表的每个位置都有两个门牌号：正向从 0 开始数（第 0 个、第 1 个…），反向从 -1 开始倒着数（倒数第 1 个、倒数第 2 个…）。

用 lst[索引] 可以取出任意位置的元素；用 lst[开始:结束] 可以切出一段。

索引与切片语法

nums = [10, 20, 30, 40, 50] nums[0] → 10 # 正向索引：第 0 个 nums[-1] → 50 # 反向索引：倒数第 1 个 nums[1:4] → [20,30,40] # 切片：取索引 1 到 3（不含 4）

动态演示：可视化索引与切片

1nums = [10, 20, 30, 40, 50]
2print(nums[1:4]) # 切片取子列表

3. 列表的增删操作 —— 队伍可以随时调整

类比：列表就像一个动态的队伍。新同学可以排到队尾（append），也可以插队到中间（insert）；有人离开时（remove / pop），队伍会自动补齐。

增删方法语法

nums = [1, 2, 3] nums.append(4) → [1, 2, 3, 4] # 追加到末尾 nums.insert(1, 99) → [1, 99, 2, 3, 4] # 在索引 1 处插入 99 nums.remove(99) → [1, 2, 3, 4] # 删除值为 99 的元素 nums.pop(0) → [2, 3, 4] # 删除并返回索引 0 的元素

动态演示：看着列表"动"起来

1nums = [1, 2, 3]
2nums.append(4) # 追加到末尾
3nums.insert(1, 99) # 在索引1处插入
4nums.remove(99) # 按值删除
5popped = nums.pop(0) # 按索引删除并取值
6
7print(nums)
8print("被弹出的值:", popped)

4. 列表的常用方法 —— 增删改查、排序统计

类比：列表自带一套万能工具箱：能往里面塞东西（append insert extend），能把东西拿出来或扔掉（pop remove clear），能把队伍排整齐（sort reverse），还能快速查找和统计（len / count / index / in）。

4.1 增加元素 —— append / insert / extend

1nums = [1, 2, 3]
2nums.append(4) # 追加到末尾
3nums.insert(1, 99) # 在索引1处插入99
4nums.extend([5, 6]) # 合并另一个列表
5print(nums)

4.2 删除元素 —— pop / remove / clear

1nums = [10, 20, 30, 40, 50]
2last = nums.pop() # 弹出最后一个
3second = nums.pop(1) # 弹出索引1的元素
4nums.remove(10) # 按值删除第一个匹配
5nums.clear() # 清空整个列表
6print(nums, "被弹出:", last, second)

4.3 排序与反转 —— sort / reverse

1nums = [3, 1, 4, 1, 5]
2nums.sort() # 从小到大排序
3print(nums)
4nums.sort(reverse=True) # 从大到小排序
5print(nums)
6nums.reverse() # 反转顺序
7print(nums)

4.4 统计与查找 —— len / count / index / in

1nums = [3, 1, 4, 1, 5, 9, 2]
2print(len(nums)) # 列表长度
3print(nums.count(1)) # 1出现了几次？
4print(nums.index(4)) # 4在第几个位置？
5print(5 in nums) # 5在列表里吗？
6print(10 not in nums) # 10不在列表里吗？

4.5 列表的加法与乘法 —— 合并与重复

1a = [1, 2, 3]
2b = [4, 5]
3print(a + b) # 加法：合并两个列表
4print(a * 3) # 乘法：重复列表3次
