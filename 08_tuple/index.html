元组 —— 不可变的数据保险箱

1. 什么是元组 —— 封好的数据包裹

类比：元组就像一个封好的快递箱 —— 里面的东西被牢牢固定住。你可以打开看（索引）、描述里面有什么（统计），但不能往里面塞新东西或者扔掉里面的东西。

用圆括号 ( ) 包起来，逗号分隔，就创建了一个元组。甚至可以不写括号，只用逗号！

基本语法

# 创建元组的几种方式 nums = (1, 2, 3, 4, 5) # 数字元组，圆括号包起来 colors = ("red", "green", "blue") # 字符串元组 mixed = (1, "hello", 3.14) # 不同类型混搭 no_parens = 1, 2, 3 # 不加括号也可以！ single = (42,) # 单个元素要加逗号，否则就是整数

⚠️ 特别提醒：创建单个元素的元组时，必须加一个逗号 (42,)，否则 Python 会认为只是一个普通的数 42！

动态演示：元组的"组装"过程

1nums = (10, 20, 30, 40, 50)
2colors = ("红", "绿", "蓝")
3no_parens = 1, 2, 3
4single = (42,)
5
6print(nums)
7print(colors)
8print(no_parens)
9print(single)

2. 元组的访问 —— 精准找到每一个元素

类比：每个元素在元组里都有两个定位编号：正向从 0 开始数（第 0 个、第 1 个…），反向从 -1 开始倒着数（倒数第 1 个、倒数第 2 个…）。跟列表完全一样！

用 t[索引] 取出任意位置的元素；用 t[开始:结束] 切出一段子元组。

索引与切片语法

nums = (10, 20, 30, 40, 50) nums[0] → 10 # 正向索引：第 0 个 nums[-1] → 50 # 反向索引：倒数第 1 个 nums[1:4] → (20,30,40) # 切片：取索引 1 到 3（不含 4）

动态演示：可视化索引与切片

1nums = (10, 20, 30, 40, 50)
2print(nums[1:4]) # 切片取子元组

3. 元组的不可变性 —— 封箱之后不能改

类比：元组就像一个密封的罐头 —— 出厂时里面有什么，就一直有什么。你不能往罐头里加东西（没有 append），也不能从里面取出东西（没有 pop），更不能修改罐头上的标签（t[0] = 新值 会报错）。

这就是元组最核心的特性 —— 不可变性。数据一旦存进元组，就永远安全不变。想要"改"？只能重新创建一个新元组。

看看"不能改"是什么意思

1t = (1, 2, 3)
2t[0] = 100 # ← 这会报错！元组不能修改
3print("这行不会执行到")

对比项列表 (List)元组 (Tuple)
符号[ ]( )

常用场景购物车、待办清单、动态数据坐标、颜色值、数据库记录、函数返回多值

4. 元组的运算 —— 合并、重复、查找、统计

类比：元组虽然不能修改内容，但你可以用它们创造新的元组！把两个元组粘起来（+），把内容复制几份（*），检查某个值在不在里面（in），或者统计一下长度和出现次数。

4.1 加法与乘法 —— 合并与重复

1a = (1, 2, 3)
2b = (4, 5)
3print(a + b) # 加法：合并两个元组
4print(a * 3) # 乘法：重复元组3次

4.2 成员判断与统计 —— in / not in / len / count / index

1nums = (3, 1, 4, 1, 5, 9, 2)
2print(len(nums)) # 元组长度
3print(nums.count(1)) # 1 出现了几次？
4print(nums.index(4)) # 4 在第几个位置？
5print(5 in nums) # 5 在元组里吗？
6print(10 not in nums) # 10 不在元组里吗？

5. 综合演示 —— 用元组管理考试成绩

用元组存储一组不可修改的考试成绩，体验 创建 → 索引 → 统计 → 成员判断 → 解包 的完整流程。

1scores = (92, 88, 95, 78) # 考试成绩（只读！）
2avg = (scores[0] + scores[1] + scores[2] + scores[3]) / 4 # 算平均分
3print("平均分:", avg)
4print("人数:", len(scores))
5has_full = 100 in scores
6print("有满分的?", has_full)
7
8# 元组解包：把元组"拆开"赋给多个变量
9student = ("小明", 18, 92)

11print(f"{name}: {age}岁, {score}分")
