函数 —— 程序的"积木"

1. 函数的定义 —— "写菜谱"

生活类比

想象你在写一本菜谱：

写菜谱 = def 定义函数 → 只是记下来，还不会做菜

按菜谱做菜 = 调用 函数 → 真正执行里面的步骤

def = 写菜谱，不写就不能做；光写不做也没用！

基本语法

def 函数名(): # def + 名字 + 冒号 """文档字符串（可选）""" # 说明函数用途 执行语句1 # 函数体（缩进4空格） 执行语句2 # 可以有多行 return 结果 # 可选：返回结果

动态演示：定义 ≠ 执行

2. 函数的调用 —— "按菜谱做菜"

生活类比

菜谱写好了，只有当你说"按这个菜谱做！"时，厨房才会真的动起来。

函数名 + 括号 = 触发执行！

调用语法

函数名() # 调用无参数函数 函数名(参数1, 参数2) # 调用带参数函数 变量 = 函数名() # 接收返回值

动态演示：定义 → 调用 → 执行

3. 参数传递 —— "给菜谱加食材"

生活类比

同一道菜，食材不同，味道不同。

def 炒菜(食材): → 菜谱写"食材"这个占位符，做菜时再决定用西红柿还是鸡蛋。

参数 = 函数的"输入口"，让函数更灵活！

参数语法

def 函数名(参数1, 参数2): # 定义时声明参数（形参） print(参数1, 参数2) 函数名("你好", 42) # 调用时传入具体值（实参） def 函数名(参数=默认值): # 带默认值的参数 print(参数)

动态演示：参数是怎么传进去的

4. 返回值 —— "上菜！"

生活类比

做完了菜 → 要端出来给人吃！

return = 把函数的结果交还给调用者。

没有 return → 函数默默做完，什么也不给。

return 语法

return 值 # 返回一个值，函数结束 return 值1, 值2 # 返回多个值（打包成元组） return # 不返回值，仅结束函数

代码示例：计算面积

def calc_area(width, height): """计算矩形面积""" area = width * height return area result = calc_area(10, 5) print(f"面积是{result}") # 输出：面积是50

动态演示：值是怎么"回来"的

5. 综合演示 —— 完整的函数之旅

把定义、调用、参数、返回值组合起来，制作一个完整的判断年龄是否成年的程序：

def is_adult(age): """判断是否成年""" if age >= 18: return True else: return False age = 20 if is_adult(age): print(f"{age}岁是成年人") else: print(f"{age}岁是未成年人")
