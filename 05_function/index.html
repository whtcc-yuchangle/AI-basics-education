函数 —— 封装代码，随叫随到

1. 函数的定义 —— 给一段代码起个名字

def 就是教会电脑一项新技能，给它起个名字、写好步骤。

就像你写下"做广播体操"的步骤清单——写下来 ≠ 马上做，只是先记好。

基本语法

def 函数名(): # def + 名字 + 冒号 """文档字符串（可选）""" # 说明函数用途 执行语句1 # 函数体（缩进4空格） 执行语句2 # 可以有多行 return 结果 # 可选：返回结果

动态演示：定义时，什么都不会发生

1def greet():
2 print("你好，世界！")
3 print("欢迎学习 Python 函数")

2. 函数的调用 —— 喊一声"去做！"

教会了技能，还得喊一声它才会动！

函数名 + () = 触发执行。不写括号，函数就只是"记得"而已。

调用语法

函数名() # 调用无参数函数 函数名(参数1, 参数2) # 调用带参数函数 变量 = 函数名() # 接收返回值

动态演示：调用才执行

1def greet():
2 print("你好！")
3 print("欢迎学习函数！")
4
5greet()

3. 参数传递 —— 告诉函数具体信息

教电脑"跟某人打招呼"时，得告诉它跟谁打——这就是参数。

def greet(name): → name 是占位符（形参），调用时再填入具体的人（实参）。

参数语法

def 函数名(参数1, 参数2): # 定义时声明参数（形参） print(参数1, 参数2) 函数名("你好", 42) # 调用时传入具体值（实参） def 函数名(参数=默认值): # 带默认值的参数 print(参数)

动态演示：参数是怎么传进去的

1def introduce(name, age):
2 print(f"{name}今年{age}岁")
3
4user_name = input("请输入名字：")
5user_age = int(input("请输入年龄："))
6introduce(user_name, user_age)

4. 返回值 —— 拿回计算结果

函数干完活，得把结果交出来！就像你问计算器"3+5等于多少"，它回答"8"。这个"8"就是返回值。

return = 函数的"交货单"。没有它，函数默默做完，你什么也拿不到。

return 语法

return 值 # 返回一个值，函数结束 return 值1, 值2 # 返回多个值（打包成元组） return # 不返回值，仅结束函数

动态演示：值是怎么"回来"的

1def calc_area(width, height):
2 area = width * height
3 return area
4
5w = float(input("宽："))
6h = float(input("高："))
7result = calc_area(w, h)
8print(f"面积是{result}")
